放假这两天好像要做英语听力测试。听力还要去网上下载，所以一时手痒，就准备折腾个播放器玩玩。

本来打算从头开始做，不过既然发现了 Kunr 的博客 上有 电台 。就去要了个源码。
代码是托管在 Github 上面的。用了 HTML5 的 Audio API，JavaScript+jQuery 控制播放器，CSS3 样式。

因为默认的样式已经比较符合审美了（其实就是懒吧！摔桌。）
所以目前看起来样式和 Kunr 的电台几乎是一模一样的 -_-#
别扭的地方做了一点修改。
UI 上有很多的细节修改。但是可能不太看得出来 XD。
比如调整了播放列表项目的宽高，菜单动画，在此不表。

Kunr 的电台用了 AJAX 从一个虾米音乐信息获取站点获取数据。然后再加载。
虽然很喜欢虾米音乐。

很显然我要删掉它。

然后，就是要导入播放列表。
可是，名字很多中文，格式比较统一，重复内容比较多。
手动复制粘贴这种事情是在是反人类，所以除了一开始尝试的版本。后续版本的播放列表都是用一段可定制的 Javascript 代码自动生成的，如下

var playlist = [];
var playlistPrefix = '14 年 PETS-2 模拟考场（';
var playlistEndfix = '）';
var playlistType = ["试题", "长对话", "短对话", "独白", "语音点拨", "13 年 9 月 PETS2 真题", "14 年 3 月 PETS2 真    题"];
var playlistTypeCount = [38, 9, 9, 3, 1, 1, 1];
var ChineseNumber = ['一', '二', '三', '四', '五', '六', '七', '八', '九'];
var i, j, listCountNum = '', playlistItem = '';
for (i in playlistType) {for (j = 0; j < playlistTypeCount[i]; j++) {listCountNum = playlistTypeCount[i] < 10 ? ChineseNumber[j] : j+1;
    listCountNum = playlistTypeCount[i] === 1 ? '' : listCountNum;
    playlistItem = playlistPrefix + playlistType[i] + listCountNum + playlistEndfix;
    playlist.push(playlistItem);
    }
}
